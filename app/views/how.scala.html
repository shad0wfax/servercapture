Our platform - Customer Support as a Service

Sign up!

When you sign up, you will be taken to the Dashboard page. You will be automatically assigned an "API KEY" (located in the Administration tab of Dashboard).
You can perform the following tasks to get started quickly:

  • Create a New Funnel
  • Install Funnel Javascript (for Web Engagement only)
  • Add Services (optional)
  • Set up Notifcations, Rules, and other settings

Create A Funnel

  • To get started you have to create a Funnel for your website / service.
  • Funnel helps you choose the way in which you want to engage with your customers. You might want certain mechanisms turned on or off.
  • Creating a new Funnel will give you basic Javascript code that you can then install on your website.
  • If you want to be only use Mobile Engagement, you need not install Javascript. We encourage a full experience for your customers and recommend Web Engagement as well.

Install Funnel Javascript

Creating a New Funnel will give you a simple Javascript code. Install it on your website.

A Sample is shown here below, but it is recommended you generate one using Funnel instead of using this.

<script type="text/javascript">
   var _visualrendezvousconfig = {
      apiKey: 'PROJECT_API_KEY_FROM_ADMIN_SCREEN',
      highlight: true,
      photo: true,
      speech2text: true,
      audio: true
      valign: 'bottom',
      halign: 'right',
      lang: 'en'
   }; 
   (function() {
	  var s = document.createElement('script');
	  s.type = 'text/javascript';
	  s.async = true;
	  s.src = '//js.visualrendezvous.com/vrendezvous.js';
	  var x = document.getElementsByTagName('head')[0];
	  x.appendChild(s);
   })();
</script>

Choose Services

We offer some out of the box services for your company to get started. As we perceive it, we want to be one single place for your entire customer interaction. You can also configure your funnel elements to automatically show up on the services that you choose.

Services combined with Funnel are necessary and sufficient systems for a Startup to have a Customeer Support Platform.

Some out of the box services we offer (not all encompassing:)

  • Discussion Forums
  • Feedback Forums
  • Knowledge Base

Dashboard and Administration

Once you have performed the tasks as described above, you are ready to act on the feedback/issues gathered. The Administration page helps you perform several tasks that makes our product a very effective web based Helpdesk.

A glimpse of things you can do from the Dashboard, including Administration:

  • Review, follow up, and act on every feedback/issue gather via any funnel element.
  • Setup Notifications on Administration page, in order to be alerted/notified as you wish. You can choose to be notified in several ways (email, sms...)
  • User management - Manage customer support personnel. Each get their own Dashboard. You can measure the overall status and even individual status./li>
  • Setup Rules - You think certain Funnels are more important. Get notified with varying degree of priority.
  • Learn from Analytics what is important to you and what your customers think are important.
